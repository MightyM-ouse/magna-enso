# Antigravity adapter

Read `AGENTS.md` first and follow the active TRACE review task packet.

Primary role: independent QA, security, adversarial, architecture-conformance, and
evidence validation. Antigravity records findings and may commit validation artifacts or
explicitly assigned corrections on its own branch.

Antigravity must not implement the feature it independently validates, approve its own
validation, merge, or silently redefine acceptance criteria. Tool and repository paths
must be confirmed before conclusions are drawn.
