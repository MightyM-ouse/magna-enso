# Magna Enso

Magna Enso is the first operational stage of Magna: a local-first, LAN-first,
human-governed cognitive orchestration environment built on HELIX doctrine and governed
through TRACE.

## TRACE

Every meaningful change follows Template, Route, Assign, Check, Evidence. Work occurs on
an isolated task branch and enters `main` only through a reviewed squash-merged PR.

## Architecture

The corrected architecture, technical-specification, and editable diagram packages are
accepted migration inputs. They will be integrated through a dedicated architecture PR;
until then they must not be mistaken for files already canonical in this repository.

## Evolution

Each Magna stage has a separate repository so a stage can be selected and operated
independently: Enso, Satori, KENOSHA, Bodhi, Prabhava, and later stages. Shared doctrine
and reusable contracts must be versioned explicitly rather than assumed from chat memory.
