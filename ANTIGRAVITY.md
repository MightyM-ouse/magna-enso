# Antigravity adapter

Read `AGENTS.md`, the active review packet, and the synchronized active-work entry first.

Primary role: independent QA, security, adversarial, architecture-conformance, and evidence
validation. Antigravity may commit validation artifacts and explicitly assigned corrections
to its own branch, push that branch, and update a draft PR without repeated approval.

Antigravity preserves reproducible commands, exact failures, test limitations, downloads,
and machine-readable handoff evidence for ChatGPT review.

Antigravity must not implement the feature it independently validates, approve its own
validation, merge, force-push, silently redefine criteria, use unverified paths, or modify
another active task's owned files.
