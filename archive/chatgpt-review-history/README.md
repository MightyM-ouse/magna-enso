# ChatGPT Review History Archive

This directory is a historical, noncanonical archive created by GOV-004. It preserves
safe project review material and provenance; it does not promote any archived claim.

## Authority boundary

GitHub `main`, Event Horizon decisions, accepted specifications, active task packets,
current code and tests, TRACE status, and curated evidence retain their normal authority.
Archive presence does not prove correctness, acceptance, implementation, validation,
release, or current status.

Conflicts must be reported. Files in this archive never silently override canonical
sources. Promotion requires a separate GitHub issue, task packet, evidence, review, and
Product Owner approval.

## Layout

- `content/` preserves the safe source hierarchy. Exact local machine paths in UTF-8 text
  were replaced using the deterministic GOV-004 token mapping.
- `_migration/GOV-004_SOURCE_MANIFEST.json` records every source filesystem object,
  hashes, disposition, redaction, duplicates, and archive mapping.
- `_migration/GOV-004_MIGRATION_REPORT.md` records execution and validation evidence.
- `_migration/GOV-004_HELD_ITEMS_SUMMARY.md` records excluded categories without secret
  values or unsafe content.

Held or excluded source objects were not copied. The local source was not modified or
deleted.
