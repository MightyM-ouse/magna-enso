# Magna ChatGPT Mentor Templates

Attach these files when asking ChatGPT to prepare prompts, review agent output, or mentor next steps for Magna / HELIX / Magna Enso.

## Files

1. `CHATGPT_MENTOR_ROLE_AND_OPERATING_MODEL.md`
   - Defines ChatGPT's role as senior architect, mentor, prompt architect, and review council.

2. `CHATGPT_PROMPT_RESPONSE_FORMAT.md`
   - Defines how ChatGPT must answer when Vinay asks for a prompt.

3. `CHATGPT_REVIEW_AND_NEXT_STEP_FORMAT.md`
   - Defines how ChatGPT must review agent output and teach Vinay from the result.

## How Vinay Uses This

For prompt creation:

```text
Prepare prompt for: <task>
```

For review:

```text
Review the attached output and provide verdict.
```

Attach these template files together with the relevant project files.
