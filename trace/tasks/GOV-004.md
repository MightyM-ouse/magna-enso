## Security and privacy gate

Run Gitleaks or the repository-approved equivalent against the source before copying, using
redacted output stored outside the repository. Never print, commit, paste, or summarize a
secret value.

Automatically hold:

- `.env` and `.env.*`, except an explicitly synthetic `.env.example`
- private keys, certificates containing private keys, tokens, credentials and password files
- cookies, browser profiles, session stores and authentication caches
- files matching secret findings
- `.git/`, `node_modules/`, virtual environments, caches and OS/editor noise
- symlinks and anything reached only through a symlink
- unreadable or special filesystem objects

A held file remains untouched in the local source. The manifest records only its relative
path, safe metadata, disposition and reason code.
